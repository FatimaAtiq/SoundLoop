<h3>Description</h3>
<p>SoundLoop is a modern music-streaming mobile application designed to deliver a smooth, intuitive listening experience inspired by platforms such as Spotify, YouTube Music, and SoundCloud. The app aims to provide seamless playback, curated playlists, personalized recommendations, and a visually engaging UI. This project will showcase skills in frontend mobile development, backend integration, music data handling, UI/UX design, and user authentication.</p>

<h3>Objectives</h3>
<ul>
  <li>Build a complete mobile application for music streaming using modern technologies.</li>
  <li>Create a clean and responsive UI with smooth transitions.</li>
  <li>Implement user authentication (sign-up, login, password recovery).</li>
  <li>Integrate music playback (play/pause, next/previous, seek, repeat, shuffle).</li>
  <li>Develop dynamic playlists, trending sections, and personalized suggestions.</li>
